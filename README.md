# Numerical Simulation of Pattern Formation

大学院の研究で実装した、反応拡散系（SDBモデル等）の数値シミュレーションプログラムです。
偏微分方程式を離散化し、C言語を用いてゼロから数値計算を行っています。

## 使用技術
- **言語:** C言語
- **可視化:** Gnuplot
- **工夫した点:** 計算結果に終わりをつけるため変化が最小になったら止める関数を設置しています。
